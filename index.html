Proceso de instalación

  1. Instalar/Actualizar el paquete VIRTUALENV

    sudo apt-get install python-pip pip show virtualenv sudo pip install --upgrade virtualenv pip show virtualenv

  2. Crear y activar un VIRTUALENV para el proyecto

    mkdir -p virtualenv cd virtualenv virtualenv --no-site-packages . source bin/activate

  3. Descargar el repositorio

    git clone git@gitlab.stic.ull.es:investigacion/portal-de-investigacion.git

  4. Instalar dependencia (desde la raíz del proyecto y con el virtualenv activo)

    pip install -U distribute sudo apt-get install libmysqlclient-dev postgresql postgresql-client postgresql-contrib libpq-dev libxml2-dev libxslt1-dev python-dev pip install -r requirements.txt

  5. Configurar PostgreSQL

  6. Cambiar contraseña del usuario postgres

    sudo su postgres -c psql ALTER USER postgres WITH PASSWORD '';

  7. Cambiar contraseña del usuario postgres en el sistema

    sudo passwd -d postgres sudo su postgres -c passwd

  8. Crear usuario y base de datos

    sudo su postgres -c psql CREATE USER investigacion WITH PASSWORD ''; ALTER USER investigacion createdb; CREATE DATABASE investigacion ENCODING = 'UTF8' TABLESPACE = pg_default LC_COLLATE = 'es_ES.UTF-8' LC_CTYPE = 'es_ES.UTF-8' CONNECTION LIMIT = -1; GRANT CONNECT, TEMPORARY ON DATABASE investigacion TO public; GRANT ALL ON DATABASE investigacion TO investigacion;

  9. Probar el acceso

    psql -U investigacion -W investigacion

  10. Crear el esquema de la base de datos

    python manage.py makemigrations python manage.py migrate

  11. Configurar Settings Local
