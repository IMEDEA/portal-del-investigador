Web Services

WS_SERVER_URL contiene la dirección URL base de los Web Services, donde está servidor donde se encuentra cada uno de los métodos. Debe sobreescribirse con la URL donde cada universidad ha puesto su servidor.

Información para el perfil de usuario

  • WS_COD_PERSONA

Este Web Service retorna un código único de tipo entero por cada investigador. Este código se guarda en el perfil de usuario, y se usa para identificar al investigador en el resto de Web Services.

VARIABLE VALOR DESCRIPCIÓN
WS_COD_PERSONA WS_SERVER_URL + 'get_codpersona?nif=' Retorna un código único que identifica al usuario en el resto de Web Services.
Ejemplo de salida del Web Service: 
33184

Generación de CVNs

Estos web services son usados para generar un CVN básico para el investigador, con la información que la universidad ya conoce sobre el mismo. De esta manera el investigador no necesita crear su CVN desde cero. Son necesarios para el funcionamiento de la opción: "Exportar mi información ULL".

  • WS_ULL_LEARNING

Este Web Service contiene información sobre la formación académica recibida por el investigador en la universidad.

VARIABLE VALOR DESCRIPCIÓN
WS_ULL_LEARNING WS_SERVER_URL + 'get_formacion_academica?cod_persona=' Formación académica recibida por el investigador 
[{"des1_titulacion": ..., 
  "des1_grado_titulacion": ...,
  "organismo": ..., 
  "f_expedicion": ...
 }, 
 {"des1_titulacion": ...
  "des1_grado_titulacion": ...,
  "organismo": ..., 
  "f_expedicion": ...
 },
 ...
]
CLAVE DESCRIPCIÓN VALOR
des1_titulacion Nombre oficial de la titulación o título del programa de doctorado Ej: 'Licenciado en Matemáticas'
organismo Entidad que expide el título o que otorga el título de doctor Ej: 'Universidad de La Laguna'
des1_grado_titulacion Tipo de titulación obtenida 'Doctor', 'Licenciado/Ingeniero Superior' o 'Diplomado/Ingeniero Técnico'
f_expedicion Fecha en la que se obtuvo el título Formato de la fecha: 'DD-MM-YYYY'
  • WS_ULL_CARGOS

Este Web Service contiene información sobre los cargos que ha ejercido el investigador en la universidad.

VARIABLE VALOR DESCRIPCIÓN
WS_ULL_CARGOS WS_SERVER_URL + 'get_cargos?cod_persona=' Cargos desempeñados por el investigador en la universidad 
[{
  "dedicacion": ..., 
  "des1_cargo": ..., 
  "centro": ..., 
  "departamento": ..., 
  "f_hasta": ..., 
  "f_toma_posesion": ...
 },
 {"dedicacion": ..., 
  "des1_cargo": ..., 
  "centro": ..., 
  "departamento": ..., 
  "f_hasta": ..., 
  "f_toma_posesion": ...
 },
 ...
]
CLAVE DESCRIPCIÓN VALOR
dedicacion Tipo de horario 'Tiempo Completo' o 'Tiempo parcial'
des1_cargo Nombre del cargo desempeñado Ej: 'Secretario del Departamento de Análisis Matemático'
centro Centro (facultad, escuela, instituto, etc.) donde está o estuvo contratado Ej: 'Departamento de Análisis Matemático'
departamento Nombre del departamento en el que estuvo adscrito en el contrato Ej: 'Análisis Matemático'
f_toma_posesion Fecha de inicio de la actividad Formato de la fecha: 'DD-MM-YYYY'
f_hasta Fecha de finalización de la actividad Formato de la fecha: 'DD-MM-YYYY'
